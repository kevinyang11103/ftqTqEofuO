# 音乐管理系统 java753

## 项目概述

音乐管理系统是基于Java语言的SpringBoot框架开发，前端采用Vue2技术，实现前后端分离的架构。系统集成了MyBatis作为持久层框架，使用MySQL数据库进行数据存储。

## 技术栈

- 后端：Java、SpringBoot、MyBatis
- 前端：Vue2
- 数据库：MySQL

## 运行环境

- 后端：JDK 1.8 或以上版本
- 前端：现代浏览器
- 数据库：MySQL 5.7 或以上版本

## 部署步骤

部署步骤详细说明如下：

1. 环境准备：确保所有依赖的软件环境已正确安装。
2. 数据库部署：执行数据库初始化脚本，建立所需数据库和表。
3. 后端部署：配置好应用.properties文件后，通过mvn或直接运行jar包启动后端服务。
4. 前端部署：在配置好代理等必要设置后，通过npm或yarn运行前端项目。
